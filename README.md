# TitanjumNote

### 超简单的笔记，支持搜索，Material Design

* 风格 square代码风格；
* 新增 支持滑动退出；
* 新增 支持单列，双列切换；
* 新增 打开 Android 等干货条目页面；
* 新增 精美背景图；
* 新增 支持长按删除；
* 特别 支持搜索内容；
* 未来：支持印象笔记、添加动画、同步等；

------

### 您将学习到：
* RecyclerView、CardView 的基本使用
* CircleImageView 的基本使用
* butterknife 的使用
* changeloglib 的使用
* 多渠道打包、友盟配置、友盟更新、友盟分享
